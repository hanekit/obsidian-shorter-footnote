# Shorter Footnote

<p align="center">
  English |
  <a href="./README_CN.md">中文</a>
</p>

在 Obsidian 中，重复出现的引用会被标注为 [num-count] 的形式，这会和 [num to num] 的形式混淆。

本插件就是为了解决此问题制作的。

---

Better Footnote 插件会影响该插件的作用，如果需要避免，请遵守以下步骤：

1. 启用 Shorter Footnote 插件
2. 关闭 Better Footnote 插件
3. 启用 Better Footnote 插件

这样两者都可以正常使用。
